Survive Adventure Game

The game "Survive" I built using visual BASIC and forms. The objective of the game is to escape the island that you crash landed on. Please note that this is a very rough game and may not work with all systems as it was built a while ago and has many compatability issues due to usage of the outdated Windows Media Player SDK. After downloading, extract the jar file and launch the game from the Survive.exe file.
